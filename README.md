# Digital Dreams Inc. — PHP / Vanilla JS Edition

Premium multi-page SaaS platform for a web design agency, engineered to run on **plain PHP shared hosting** (cPanel / Plesk) — no Node.js, no Composer.

Built with:
- **PHP 8** + PDO + MySQL (25-table schema, auto-seeded)
- **Vanilla JavaScript** + HTML5 (no build step)
- **Custom CSS** (Aurora glow, shimmer text, 3D tilt, glass cards) — Apple-style, no framework
- **Cashfree** payments (with safe mock-mode fallback)

## Features

- Multi-section landing page (hero • services • features • packages • add-ons • portfolio • why-choose • process • testimonials • blog • FAQ • contact)
- 10-step checkout with live pricing, coupons, uploads and Cashfree hand-off
- Customer dashboard: orders, invoices, payments, tickets, messages, downloads, uploads, profile
- Admin panel: 24+ sections including full CRUD for packages, add-ons, coupons, testimonials, portfolio, blog, users, customers, leads, email templates and site settings
- Automated email templates + PHP `mail()` sending (stub mode when disabled)
- Printable invoices
